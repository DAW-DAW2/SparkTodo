# Spark proyect running on tomcat container

###tags
*sparkjava, tomcat7, gradle, angularjs, bootstrap, mongodb*

        gradle war

Run on embedded tomcat server
        
        gradle tomcatRunWar

deploy to tomcat

##Requisites
- Jdk 1.8
- MongoDB

##Configuration

Tomcat must use jdk1.8

> *Set JAVA_HOME in /etc/default/tomcat7*
